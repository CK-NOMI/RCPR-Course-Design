## 5. 代码主入口

当前最重要的推理入口是：

- `test.py`：生成预测图
- `eval_metrics.py`：评测已有预测图
- `run_aqua20_u1.ps1`：对 AQUA20 做图片推理

### 5.1 `test.py` 的核心逻辑

`test.py` 会：

1. 从 `./checkpoints/<checkpoint_name>` 加载权重
2. 从 `./datasets/<dataset>/image` 和 `./datasets/<dataset>/groundtruth` 读取数据
3. 把输出写入 `./predictions/<model_folder>/`

它当前支持：

- `--stage2_proto mean`
- `--stage2_proto acre`
- `--topk_mode legacy`
- `--topk_mode rtg`
- `--rpf_rounds 1`
- `--rpf_rounds 2`

## 6. 标准数据集推理

### 6.1 跑三套标准测试集

这是最常用的主命令。当前推荐直接用一行：

```powershell
cd C:\COSOD\SCoSPARC-main课设版 && python test.py --model_folder ours_main_run --checkpoint_name model_combo_base8-136_0.7291838924090067.pt --datasets all --size 224 --test_num_workers 8 --max_group_images 0 --stage2_proto acre --topk_mode rtg --topk_ratio 0.04 --topk_res_alpha 0.10 --topk_conf_gate 0.58 --topk_mass_min 0.45 --topk_delta_th 0.045 --rpf_rounds 2 --rpf_soft_lambda 0 --tau2_mode fixed --tau2_delta 0.005 --baseline_legacy 0 --tau1_sim 0.76
```

说明：

- `--datasets all` 会依次跑 `CoCA`、`Cosal2015`、`CoSOD3k`
- 输出会写到 `predictions\ours_main_run\`
- 这是当前项目后期常用的 ACRE + RTG-TopK + RPF 配置

### 6.2 只跑 CoCA

```powershell
cd C:\COSOD\SCoSPARC-main课设版 && python test.py --model_folder ours_coca_only --checkpoint_name model_combo_base8-136_0.7291838924090067.pt --datasets CoCA --size 224 --test_num_workers 8 --max_group_images 0 --stage2_proto acre --topk_mode rtg --topk_ratio 0.04 --topk_res_alpha 0.10 --topk_conf_gate 0.58 --topk_mass_min 0.45 --topk_delta_th 0.045 --rpf_rounds 2 --rpf_soft_lambda 0 --tau2_mode fixed --tau2_delta 0.005 --baseline_legacy 0 --tau1_sim 0.76
```

### 6.3 只跑 ACRE 单轮版本

如果只是想跑较稳的 ACRE 单轮版本，不开 RPF：

```powershell
cd C:\COSOD\SCoSPARC-main课设版 && python test.py --model_folder ours_acre_only_t076 --checkpoint_name model_combo_base8-136_0.7291838924090067.pt --datasets all --size 224 --test_num_workers 8 --max_group_images 0 --stage2_proto acre --topk_ratio 0 --rpf_rounds 1 --rpf_soft_lambda 0 --tau2_mode fixed --tau2_delta 0 --baseline_legacy 0 --tau1_sim 0.76
```

## 8. AQUA20 图片推理

### 8.1 当前脚本在做什么

当前课设版已经有：

`run_aqua20_u1.ps1`

这个脚本不是直接新增一个新数据集名，而是采用了“临时把 `datasets\CoCA` 指向 `datasets\AQUA20_grouped`”的办法来复用 `test.py` 现有逻辑。

具体逻辑是：

1. 备份原始 `datasets\CoCA`
2. 创建一个指向 `datasets\AQUA20_grouped` 的 junction
3. 用 `--datasets CoCA` 跑推理
4. 结束后恢复原始 `datasets\CoCA`

所以：

- 这是一个“路径映射方案”
- 不要在脚本运行中手动打断并改动 `datasets\CoCA`
- 如果中途异常退出，要检查 `datasets\CoCA` 是否已经恢复

### 8.2 直接运行 AQUA20 脚本

推荐命令：

```powershell
cd C:\COSOD\SCoSPARC-main课设版 && powershell -ExecutionPolicy Bypass -File .\run_aqua20_u1.ps1
```
